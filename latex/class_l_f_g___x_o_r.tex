\doxysubsection{Detailed Description}
Генератор Фибоначчи с запаздыванием и XOR (LFG-\/\+XOR). 

Базовый алгоритм:\+ LFG с лагами (55,24). Модификации:\+
\begin{DoxyItemize}
\item Вместо сложения используется XOR (быстрее, лучше перемешивание).
\item Добавлена нелинейная функция scramble после обновлений.
\item Улучшенная инициализация буфера через LCG и перемешивание. 
\end{DoxyItemize}
