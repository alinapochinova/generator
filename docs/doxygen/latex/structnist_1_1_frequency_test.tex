\doxysubsection{Detailed Description}
Частотный (monobit) тест — проверяет, близка ли доля единиц во всей последовательности к 1/\+2. 

Статистика:\+ S\+\_\+n = (количество единиц) -\/ (количество нулей). Нормированная статистика:\+ S = \texorpdfstring{$\vert$}{|}\+S\+\_\+n\texorpdfstring{$\vert$}{|} /\+ sqrtn. p-\/value = erfc(S/\+sqrt2). Если p-\/value ge alpha, последовательность случайна.
