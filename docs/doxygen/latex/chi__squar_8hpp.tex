\doxysubsection{Detailed Description}
Реализация критерия согласия хи-\/квадрат (Пирсона) для проверки равномерности распределения. 

Содержит функции chi\+\_\+squared (вычисление статистики и степеней свободы) и chi\+\_\+squared\+\_\+test (проверка гипотезы на заданном уровне значимости).
