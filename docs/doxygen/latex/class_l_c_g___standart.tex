\doxysubsection{Detailed Description}
Классический линейный конгруэнтный генератор (LCG) без модификаций. 

Формула:\+ state = A \texorpdfstring{$\ast$}{*} state + C mod 2\texorpdfstring{$^\wedge$}{\string^}64. Параметры A и C выбраны так же, как в модифицированной версии, но без пост-\/перемешивания (XOR со сдвигами).

\begin{DoxyNote}{Note}
Период:\+ 2\texorpdfstring{$^\wedge$}{\string^}64 при правильных параметрах. 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
Младшие биты имеют короткий период, есть корреляции между соседними числами.
\end{DoxyWarning}
