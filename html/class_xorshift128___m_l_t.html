Detailed Description

Модифицированный Xorshift128 с нелинейным умножением (Xorshift128-MLT).

Базовый алгоритм: Xorshift128 (период 2^128-1). Модификации:

  • Добавлено нелинейное умножение на две константы и дополнительные XOR-сдвиги.
  • Прогрев (10 итераций) для устранения корреляции начальных значений.

Инициализирует два 64-битных слова состояния: s[0] = seed, s[1] = seed * константа (золотое сечение). Затем делается 10 итераций next() для «прогрева» — чтобы начальные значения хорошо перемешались и убрать возможную корреляцию.

Шаги:

  1. Выполняется классический Xorshift128:
    • берём s1 = s[0], s0 = s[1]
    • обновляем s[0] = s0
    • выполняем серию XOR и сдвигов: s1 ^= s1<<23; s1 ^= s1>>17; s1 ^= s0; s1 ^= s0>>26
    • сохраняем s[1] = s1
  2. Применяем нелинейное преобразование к s1:
    • умножаем на константу 0xbf58476d1ce4e5b9ULL
    • делаем XOR со сдвигом вправо на 31 бит
    • умножаем на константу 0x94d049bb133111ebULL
    • делаем XOR со сдвигом вправо на 27 бит Эти операции взяты из генератора SplitMix64 и обеспечивают высокое качество случайности, сохраняя период исходного Xorshift.
