Detailed Description

Реализация критерия согласия хи-квадрат (Пирсона) для проверки равномерности распределения.

Содержит функции chi_squared (вычисление статистики и степеней свободы) и chi_squared_test (проверка гипотезы на заданном уровне значимости).

Template Parameters

Parameters

minМинимальное возможное значение (включительно).
maxМаксимальное возможное значение (включительно для UNIFORM_INT, исключительно для UNIFORM_REAL).
typeТип распределения (целочисленное или непрерывное).
Returns

Exceptions
std::invalid_argumentесли min >= max, или размер выборки < 2, или значение выборки вне диапазона.
