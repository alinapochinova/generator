Detailed Description

Классический генератор Фибоначчи с запаздыванием (LFG) без модификаций.

Формула: buffer[i] = buffer[i] ^ buffer[(i+24)%55] (без scramble). Лаги: p=55, q=24.

Note
Период: ~2^79 для XOR-варианта.
Warning
Сильная зависимость от начального seed, возможны корреляции.
See also
LFG_XOR – модифицированная версия с scramble и улучшенной инициализацией

Простая инициализация буфера без перемешивания.

Parameters
seedНачальное значение.

Заполняет буфер из 55 элементов с помощью LCG: state = state * 1103515245 + 12345.

Note
Нет функции scramble, нет дополнительных проходов перемешивания. Это делает генератор чувствительным к выбору seed.

uint32_t LFG_Standart::next ( )

Генерирует следующее 32-битное псевдослучайное число.

Генерация следующего 32-битного числа.

Returns
Очередное значение LFG без scramble.

Если индекс достиг конца буфера (55), весь буфер пересчитывается по формуле: buffer[i] ^= buffer[(i+24)%55]. Затем возвращается текущий элемент буфера, и индекс увеличивается.

Note
Отсутствие scramble может приводить к корреляциям.
