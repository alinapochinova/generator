\doxysubsection{Detailed Description}
Модифицированный LCG с нелинейным перемешиванием (MLCG-\/\+XOR). 

Классический LCG:\+ X\+\_\+\{n+1\} = (a \texorpdfstring{$\ast$}{*} X\+\_\+n + c) mod 2\texorpdfstring{$^\wedge$}{\string^}64. Модификации:\+
\begin{DoxyItemize}
\item Модуль 2\texorpdfstring{$^\wedge$}{\string^}64 за счёт естественного переполнения uint64\+\_\+t.
\item Параметры a и c подобраны для полного периода 2\texorpdfstring{$^\wedge$}{\string^}64.
\item После каждого шага применяется нелинейное перемешивание (XOR и сдвиги), устраняющее корреляцию младших битов. 
\end{DoxyItemize}

Генерирует 64-\/битное псевдослучайное число. 

Генерирует следующее псевдослучайное 64-\/битное число.

\begin{DoxyReturn}{Returns}
64-\/битное число, равномерно распределённое в \+[0, 2\texorpdfstring{$^\wedge$}{\string^}64-\/1]\+.
\end{DoxyReturn}
Шаги:\+
\begin{DoxyEnumerate}
\item Выполняется LCG-\/шаг:\+ state = A \texorpdfstring{$\ast$}{*} state + C. Переполнение uint64\+\_\+t автоматически берёт результат по модулю 2\texorpdfstring{$^\wedge$}{\string^}64. Параметры A и C подобраны так, чтобы период был максимальным (2\texorpdfstring{$^\wedge$}{\string^}64).
\item К полученному состоянию применяются три битовые операции:\+
\begin{DoxyItemize}
\item x \texorpdfstring{$^\wedge$}{\string^}= (x \texorpdfstring{$>$}{>}\texorpdfstring{$>$}{>} 31) /\+/\+ сдвиг вправо и XOR
\item x \texorpdfstring{$^\wedge$}{\string^}= (x \texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<} 25) /\+/\+ сдвиг влево и XOR
\item x \texorpdfstring{$^\wedge$}{\string^}= (x \texorpdfstring{$>$}{>}\texorpdfstring{$>$}{>} 28) /\+/\+ ещё один сдвиг вправо Эти операции нелинейны и хорошо перемешивают биты, разрушая корреляцию младших битов, свойственную LCG. 
\end{DoxyItemize}
\end{DoxyEnumerate}
