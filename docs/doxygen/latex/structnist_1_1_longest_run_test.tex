\doxysubsection{Detailed Description}
Тест на самую длинную серию — проверяет, не является ли максимальная длина серии в блоках слишком большой (что указывает на неслучайность). 

Последовательность разбивается на блоки по 8 бит. Для каждого блока находится максимальная длина серии (подряд идущих одинаковых битов). Затем сравнивается наблюдаемое распределение максимальных длин с теоретическим. Статистика chi-\/squared рассчитывается по 7 категориям (длина le1, 2, 3, 4, 5, 6, ge7). p-\/value вычисляется приближённо как exp(-\/chi-\/squared/\+2).
