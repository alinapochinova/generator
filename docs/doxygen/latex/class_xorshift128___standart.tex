\doxysubsection{Detailed Description}
Классический Xorshift128 без нелинейного умножения. 

Формула:\+ только XOR и сдвиги (23, 17, 26). Состояние:\+ 128 бит (два 64-\/битных слова).

\begin{DoxyNote}{Note}
Период:\+ 2\texorpdfstring{$^\wedge$}{\string^}128 -\/ 1. 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
Линейность в GF(2) – по нескольким выходам можно восстановить состояние.
\end{DoxyWarning}
