Detailed Description

Модифицированный LCG с нелинейным перемешиванием (MLCG-XOR).

Классический LCG: X_{n+1} = (a * X_n + c) mod 2^64. Модификации:

  • Модуль 2^64 за счёт естественного переполнения uint64_t.
  • Параметры a и c подобраны для полного периода 2^64.
  • После каждого шага применяется нелинейное перемешивание (XOR и сдвиги), устраняющее корреляцию младших битов.

Шаги:

  1. Выполняется LCG-шаг: state = A * state + C. Переполнение uint64_t автоматически берёт результат по модулю 2^64. Параметры A и C подобраны так, чтобы период был максимальным (2^64).
  2. К полученному состоянию применяются три битовые операции:
    • x ^= (x >> 31) // сдвиг вправо и XOR
    • x ^= (x << 25) // сдвиг влево и XOR
    • x ^= (x >> 28) // ещё один сдвиг вправо Эти операции нелинейны и хорошо перемешивают биты, разрушая корреляцию младших битов, свойственную LCG.
