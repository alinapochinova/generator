\doxysubsection{Detailed Description}
Тест на серии (runs test) — проверяет, соответствует ли количество серий (последовательностей одинаковых битов) случайному распределению. 

Пусть pi = количество единиц /\+ n. Если pi не слишком отличается от 0.\+5, статистика:\+ Z = (R -\/ 2npi(1-\/pi)) /\+ (2sqrt(2n)pi(1-\/pi)), где R — общее число серий. p-\/value = erfc(\texorpdfstring{$\vert$}{|}\+Z\texorpdfstring{$\vert$}{|}/\+sqrt2). Малое p-\/value указывает на корреляцию соседних битов.
