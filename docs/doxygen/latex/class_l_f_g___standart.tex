\doxysubsection{Detailed Description}
Классический генератор Фибоначчи с запаздыванием (LFG) без модификаций. 

Формула:\+ buffer\+[i]\+ = buffer\+[i]\+ \texorpdfstring{$^\wedge$}{\string^} buffer\+[(i+24)\%55]\+ (без scramble). Лаги:\+ p=55, q=24.

\begin{DoxyNote}{Note}
Период:\+ \texorpdfstring{$\sim$}{\string~}2\texorpdfstring{$^\wedge$}{\string^}79 для XOR-\/варианта. 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
Сильная зависимость от начального seed, возможны корреляции.
\end{DoxyWarning}
