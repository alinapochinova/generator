Detailed Description

Классический Xorshift128 без нелинейного умножения.

Формула: только XOR и сдвиги (23, 17, 26). Состояние: 128 бит (два 64-битных слова).

Note
Период: 2^128 - 1.
Warning
Линейность в GF(2) – по нескольким выходам можно восстановить состояние.
See also
Xorshift128_MLT – модифицированная версия с умножением (SplitMix64)

uint64_t Xorshift128_Standart::next ( )

Генерирует следующее 64-битное псевдослучайное число.

Генерация следующего 64-битного числа.

Returns
Очередное значение Xorshift без нелинейного умножения.

Выполняется классический Xorshift128 с параметрами (23, 17, 26):

  • s1 ^= s1 << 23
  • s1 ^= s1 >> 17
  • s1 ^= s0
  • s1 ^= s0 >> 26
Note
Операции линейны в GF(2), что может приводить к корреляциям. Модифицированная версия добавляет умножение для нелинейности.
